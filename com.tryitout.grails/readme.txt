1. Install Spring Security Core:
    grails install-plugin spring-security-core
2. Configure security  (http://grails-plugins.github.com/grails-spring-security-core/docs/manual/guide/single.html#23%20Tutorials)
    grails s2-quickstart com.tryitout.grails User Role
3. Check this tutorial for Resource plugin (13:08min):
http://www.youtube.com/watch?feature=player_embedded&v=HCUJuGXZ9A0

and this blog entry: 
http://blog.springsource.org/2011/06/30/countdown-to-grails-2-0-static-resources/
http://www.anyware.co.uk/2005/2011/09/12/optimising-your-application-with-grails-resources-plugin/

You can view db content using: http://localhost:8080/com.tryitout.grails/dbconsole/
