
To run this sample type:  mvn jetty:run

Application will be available at localhost:8080/com.tryitout.resteasy/
e.g.:
1. try message service:
localhost:8080/com.tryitout.resteasy/rest/message/Hello%20World
2. try JSON service:
http://localhost:8080/com.tryitout.resteasy/rest/json/movie/get
